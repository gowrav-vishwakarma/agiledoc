Installing agiletoolkit

This section will demonstrate how to get and install agiletoolkit by the first and simplest method. Download the stable version of framwork from Stable Version of Agiletoolkit from agiletoolkit.org. But we prefer you to download beginner pack of stable release from xavoc-internataional. Download Basic Pack of Stable Release.

System Requirements

Before proceeding have a look at the following system-requirements for agiletoolkit
  • Agiletoolkit requires PHP 5.3
  • Any Web server
  • One of the following Database systems: MySQl, SQlite, MonoDB, prostSQL [Optional]

Installation http://youtu.be/5Mcryu__-uY

After downloading the agiletoolkit framework, follow the following procedure:
  1. Create a folder for your project at your document root ie. 'htdocs' in xampp or '/var/www' in ubuntu or wherever defined
  2. Extract the downloaded zip file in your own project folder
  3. Point your web browser at 'http://localhost/your_folder', if you have done everything okey, you should see your new empty agile project images/installation/installed.png
